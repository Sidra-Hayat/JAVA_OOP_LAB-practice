# JAVA OOP Lab Practice

A comprehensive Java repository dedicated to Object-Oriented Programming (OOP) concepts and practical exercises.

## 📚 Overview

This repository contains practice problems, examples, and solutions for learning and mastering Object-Oriented Programming principles in Java. It's designed for students and developers looking to strengthen their understanding of OOP concepts through hands-on coding.

## 🎯 Topics Covered

- **Classes and Objects** - Fundamentals of object creation and manipulation
- **Inheritance** - Code reusability through class hierarchies
- **Polymorphism** - Method overriding and method overloading
- **Encapsulation** - Data hiding and access modifiers
- **Abstraction** - Abstract classes and interfaces
- **Collections** - Working with ArrayList, HashMap, and other data structures
- **Exception Handling** - Try-catch blocks and custom exceptions
- **File I/O** - Reading and writing files
- **Packages** - Organizing code into logical modules

## 🚀 Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code with Java extensions) or command-line tools

### Compilation and Execution

**Using command line:**
```bash
# Compile
javac src/path/to/YourFile.java

# Run
java -cp src path.to.YourClass
```

**Using an IDE:**
1. Open the project in your preferred IDE
2. Right-click on the Java file
3. Select "Run" or "Run As Java Application"

## 💡 Key Concepts

### Object-Oriented Programming Pillars

1. **Encapsulation** - Bundle data and methods; control access through modifiers
2. **Inheritance** - Create class hierarchies to promote code reuse
3. **Polymorphism** - Write flexible code that works with multiple types
4. **Abstraction** - Hide complexity; expose only essential features
